## 🏃‍♂️ Descripción de tareas del SPRINT 2
El SPRINT 2 consta de tres partes a desarrollar, cada una de las cuales se describe a continuación:

1. **Contenido dinámico con JSON y templates**  
   - Incluir el contenido final del sitio web de forma dinámica usando vanilla Javascript.
   - El contenido se almacenará en un fichero JSON estructurado como una base de datos.
   - Este fichero JSON incluirá datos generales del sitio web y usuarios para validación y autenticación.
   - Usar templates básicos (como header y footer) para generar las páginas finales dinámicamente.

2. **Formularios con validación en cliente**  
   - Incluir al menos un formulario en alguna página del sitio web.
   - Implementar validación en el lado cliente usando HTML5.
   - Crear un formulario para registrar y autenticar usuarios.
   - Mostrar contenido diferenciado dependiendo de si el usuario está autenticado o no, o según su rol.

3. **Responsive Web Design (RWD)**  
   - Adaptar los templates actuales para dispositivos de distintos tamaños:
     - Tamaño grande (desktop y portátil).
     - Tamaño medio (tablet).
     - Tamaño pequeño (móvil).
   - Usar técnicas como media queries.
   - Verificar los diseños ajustando el tamaño de la ventana del navegador para comprobar los breakpoints.
